# Walrus Feedback

This is the official feedback hub for **[Walrus on a Zipline](https://walrus.studio)**.

Have a bug to report, a suggestion to share, or just want to say something? Open an issue — we read everything.

## How to submit feedback

1. Click the **[Issues](../../issues)** tab above
2. Click **New issue**
3. Describe your bug or suggestion and hit **Submit**

That's it. No account required beyond a free GitHub login.

## What to include in a bug report

- What happened and what you expected to happen
- What browser and device you were using
- Any steps to reproduce the issue (if you can)

Thanks for playing!
